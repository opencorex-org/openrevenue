# Roadmap

1. Foundation: vertical slice, contracts, modular boundaries, fictional country pack, and local operations.
2. Persistence: pgx/sqlc repositories, outbox publisher, OIDC adapters, document scanning, and generated clients.
3. Pilot readiness: workflow engine, configurable forms/rules, accessibility, localization, performance and security assurance.
4. Ecosystem: stable extension contracts, additional community country packs, deployment profiles, and service extraction where measured load requires it.
