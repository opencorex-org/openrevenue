# Governance

OpenRevenue is an [OpenCorex](https://github.com/opencorex-org) open-source project and uses a maintainer-led, consensus-seeking model. Maintainers review changes, releases, security responses, and architectural decisions. Material or irreversible choices require an ADR and public proposal. If consensus cannot be reached, maintainers vote by simple majority; security embargo decisions remain private until disclosure is safe. Roles and voting membership will be published as the community grows.
